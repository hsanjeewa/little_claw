# Triage labels

| Role | Label |
|---|---|
| Needs evaluation | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Ready for an AFK agent | `ready-for-agent` |
| Ready for a human | `ready-for-human` |
| Won't fix | `wontfix` |

These are the default canonical labels. Override here if the repo's label names differ.
